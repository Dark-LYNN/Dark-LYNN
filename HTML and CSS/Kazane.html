Invite Me

Prefix: k.
General Commands
• Help — Display the bot’s list of commands.
• About — Get a link to invite the bot to your server and the bot’s libraries
• Ping — Show the bot’s current response time.
• Server — Get information of this guild.

Music Commands
Play [URL|YT search] — Joins and play music in a channel.
Spotplay, [URL|Playlist] — Plays a specific Spotify link
Pause — Pause the music player.
Resume — Resume the music player.
Stop — Stop and clear the music player.
Skip — Skip the current music.
Nowplaying — Shows what’s currently playing.
Queue — Shows the music that’s currently queued.
Loop — Loops the current queue.
Volume (1-150) — Set the volume of the music player.

Admin Commands
setLog (channel) — Sets a Log channel for the music commands.
MusicLog [on , off] — Enables / Disables music logs for the server.
SetDJ (Role) — Sets a DjRole
